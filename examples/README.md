# Examples

Run any file with `uv run python examples/<path>.py`.

- **`engine/`** — the 16 patterns on the raw engine (`Node`, `System`, executor).
- **`core-sdk/`** — the typed SDK over `fedotmas`: `flow/` (arrows), `board/` (blackboard), `serialize/`, plus `running.py` and `conditions.py`.
- **`llm/`** — the same SDK with `fedotmas-llm` (needs a `.env` API key).
- **`meta/`** — presets from `fedotmas-meta`.
